# mall-admin-web

商城管理后台，基于 `Vue3 + TypeScript + Element Plus + Pinia`。

## 启动前准备

### 1. 配置接口地址

参考 `.env.example` 新建本地环境文件，例如：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8080
```

### 2. 启动开发环境

```bash
npm install
npm run dev
```

## 当前已接通的后端模块

- 工作台
- 商品分类
- 商品管理
- 订单管理
- 用户管理
- 优惠券管理
- 首页 Banner 配置
- 系统设置
